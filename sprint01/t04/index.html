CSS

Selector

The name of the HTML element at the beginning of the rule set. It selects the element (s) to style (in this case, p elements). To style another element, just change the selector.

p {

}

background-color

Specifies the background color of an element. Although this property does not inherit properties from its parent, because the initial value is set to transparent, the background color of the children is the same as the background color of the parent.

h1 {
background-color: RGB(249, 201, 16);
}

font-family

Sets the font family to be used to style the text of the content. The font list can include one or more names, separated by commas.

font-family: font name [, font name[, ...]] | inherit

opacity

Determines the transparency level of a web page element. With partial or full transparency, the background image or other elements below the translucent object appear through the element.

opacity: value;

border

The generic border property allows you to simultaneously set the thickness, style, and color of the border around an element. The values can go in any order, separated by a space, the browser will determine which of them corresponds to the desired property. To set the border only on certain sides of an element, use the border-top, border-bottom, border-left, border-right properties.

border: [border-width || border-style || border-color] | inherit

margin

Sets the padding from each edge of the element. The padding is the space from the current element's border to the inner border of its parent element

body {
margin: 20px;
}

text-align

Determines the horizontal alignment of the text within the element.

text-align: center | justify | left | right | start | end

height

Sets the height of block or replaceable elements. Height does not include the thickness of the borders around the element, padding, and margins.

height: value | percent | auto | inherit

Declaration

Rule, for example color: red; specifies which of the element's properties you want to style.

p {
color: red;
}

width

Sets the width of block or swappable elements. Width does not include the thickness of the borders around the element, padding, and margins.

height: value | percent | auto | inherit

Protocols

IP

The Internet Protocol (IP) is the principal communications protocol in the Internet protocol suite for relaying datagrams across network boundaries. Its routing function enables internetworking, and essentially establishes the Internet. IP has the task of delivering packets from the source host to the destination host solely based on the IP addresses in the packet headers. For this purpose, IP defines packet structures that encapsulate the data to be delivered. It also defines addressing methods that are used to label the datagram with source and destination information.

DNS

The Domain Name System (DNS) is a central part of the internet, providing a way to match names (a website you’re seeking) to numbers (the address for the website). Anything connected to the internet - laptops, tablets, mobile phones, websites - has an Internet Protocol (IP) address made up of numbers. Your favorite website might have an IP address like 64.202.189.170, but this is obviously not easy to remember. However a domain name such as bestdomainnameever.com is something people can recognize and remember. DNS syncs up domain names with IP addresses enabling humans to use memorable domain names while computers on the internet can use IP addresses.

HTTP/HTTPS

HTTP is a protocol that describes the rules for transmitting data on the Internet. It helps the browser load web pages and the server retrieves the information that the user has entered on the site.
HTTPS is the same protocol but with a security add-on.

Description

In networking, a protocol is a set of rules for formatting and processing data. Network protocols are like a common language for computers. The computers within a network may use vastly different software and hardware; however, the use of protocols enables them to communicate with each other regardless.
